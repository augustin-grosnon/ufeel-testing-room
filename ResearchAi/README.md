| **Human**                    | Good accuracy          | Fast          | FER2013               | TensorFlow   |
| MoodMoji                     | Average accuracy       | Slow          | DeepMoji              | TensorFlow   |
| Emotion Recognition          | Good accuracy          | Fast          | Their own data        | PyTorch      |

# Setup Instructions

## Requirements

- Unity 6+
- Webcam and microphone with access permissions
- (Optional) Biometric sensor for the heartbeat scene

## Steps

1. **Clone or download** the repository.
2. Open the project in **Unity Hub**.
3. Import the required packages:
   - UFeel
4. Open the `TestingRoom` scene and press Play to begin.

## Hardware Setup

- Grant webcam access for emotion detection (should be automatic but access right might be asked on startup).
- Make sure no other application is blocking the webcam
- For the Heartbeat scene, connect a supported biometric sensor.
